# Contributing to Dailsmart AI

Welcome to Dailsmart AI! ❤️ Thank you for your interest in contributing to the future of open-source voice AI. ❤️

Dailsmart AI is a comprehensive voice agent platform that helps developers build, test, and deploy conversational AI systems with minimal setup.

## How to Contribute

- 🐛 **Report bugs** via GitHub Issues
- 💡 **Suggest features** via GitHub Discussions
- 📖 **Improve documentation** by submitting a PR
- 💻 **Contribute code** (bug fixes, new features, new integrations)

### Getting Started

1. Fork the repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request
